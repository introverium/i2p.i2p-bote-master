O que é uma Identidade?

Uma identidade é como uma conta de email, mas sem um provider - tu controlas a tua conta, em vez de um terceiro, como Google ou Yahoo

O que é um Destino?

Um Destino, é o endereço que das aos teus amigos para que eles possam enviar-te emails

Porque é que os endereços de correio eletrónico são compridos?

Todos os emails são automaticamente encriptados pelo Bote. Cada endereço contêm um conjunto de chaves públicas, que são usadas para encriptar os emails enviados por esse endereço. O tamanho do endereço depende do typo de encriptação usado.

Existe um limite pra o número de identidades que um utilizador pode ter?

Podes ter quantas Identidades quizeres, e usar diferentes Identidades para diferentes propósitos

Quando é que as Identidades expiram?

Nunca. Desde que tenhas as chaves privadas para uma Identidade, podes usa-la. Isso também significa que não existe a possibilidade de "recuperar contas" - caso percas o telefone, perdes todas as tuas Identidades

Podes guardar as tuas Identidades usando a opção Exportar que está disponível no menu "Opções". Isto irá guardar uma cópia das tuas Identidades num ficheiro, que estará no teu telefone. Este ficheiro pode ser usado para recuperar as tuas Identidades ou para transferir para outro aparelho.
